AWS Learning Journey  
Welcome to my AWS learning repository! 🚀  

This repo contains my notes, hands‑on labs, and concepts that I’m learning while exploring Amazon Web Services (AWS). I’ll be updating it regularly as I go deeper into different AWS services.  

🎯 Goals of this Repository  
Build a strong foundation in core AWS services  
Document important concepts in simple language  
Practice using AWS through hands‑on examples  
Create a reference I (and others) can revisit later  

📂 Repository Structure  
Planned structure (may change as I learn more):  

compute/ – EC2, Lambda, Autoscaling, Elastic Beanstalk, etc.  
storage/ – S3, EBS, EFS, Storage Classes, Lifecycle rules  
VPC/ – Networking, subnets, route tables, security groups, NACLs   
iam/ – IAM users, roles, policies, permissions  
RDS/ – RDS, Aurora, backups, read replicas   
others/ – Any other services or miscellaneous notes   
  

Each folder may contain:  
notes.md – Theory and concepts   
labs/ – Step‑by‑step hands‑on exercises   
diagrams/ – Architecture diagrams  


🧠 Topics I Plan to Cover  
Some of the AWS areas I’ll focus on:  
Compute: EC2, AMIs, security groups, key pairs, Load Balancers, Auto Scaling  
Storage & Databases: S3 basics, versioning, encryption, RDS concepts    
Networking: VPC, public/private subnets, Internet Gateway, NAT Gateway  
Security: IAM basics, roles, policies, least privilege    
Monitoring: Basic CloudWatch concepts (metrics, alarms)   
High Availability & Scalability: Multi‑AZ, load balancing basics  


🧪 How to Use This Repo   
You can:   
Read the notes to understand AWS concepts in simple terms  
Follow the lab steps to practice in your own AWS account    


🛠 Prerequisites   
An AWS account (free tier)    
Basic understanding of Linux, networking, and the cloud (helpful but not mandatory)  
AWS Management Console access
